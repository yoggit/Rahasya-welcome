# Rahasya — welcome page

Public landing page for **Rahasya**, a private cryptography app for curious kids.
One static `index.html` (self-contained), served via GitHub Pages.

- **App:** https://rahasya-cipher.web.app (sign in with Google to start)
- **Subtitle:** *The Logic of Locks*
- **Tagline:** *Real cryptography for curious minds — build every cipher, then break it.*

Rahasya teaches the real standard algorithms, not invented toys. Each rung earns the next: a lock,
then why it fails, then the lock that fixes it. Every cipher ships a "Break it" mode, because you
must think like an attacker to build a good lock. This page is the parent-facing introduction.

Part of a small family of ad-free, offline-first learning apps for curious kids.
